# BiteMe Archetype Writing Guide

This document defines the tone and structure for all user taste profile descriptions generated in the application.

Any code generating archetype descriptions should follow the rules below.


Taste descriptions should feel:
- human
- observant
- dry
- slightly teasing
- concise
- not corporate
- not chatbot-like

The tone should resemble a smart friend noticing your food habits.

Think: wry observation, not punchline comedy.

## Rules

Do NOT use phrases like:

- "flavor lean"
- "cuisine gravity"
- "plate structure"
- "analysis suggests"

Avoid sounding like:
- a dashboard
- a consultant
- a personality test

Prefer:
- specific observations
- slightly dry humor
- understatement

## Structure

Each archetype should produce:

1. Archetype title
2. One short description (1–2 sentences)
3. One dry observation line
4. Optional 1–2 supporting observations

Total: **3–4 short sentences max**

## Examples

### Balanced Bistro Strategist

You’ve got range, but not the chaotic kind. Dinner rotates just enough to stay interesting without turning into a culinary identity crisis.

You like variety — just with guardrails.

Give it a few more uploads and we’ll see how firm those guardrails really are.

---

### Comfort Cartographer

You know what you like, and much of it qualifies as emotional infrastructure.

The salad appears occasionally, which is nice for everyone.

---

### Savory Maximalist

You rarely pick the timid option.

If something can be roasted, brothy, rich, or deeply unnecessary in the best way, you’re interested.

---

### Heat-Seeking Diplomat

You flirt with chaos, but in a controlled way.

Spice isn’t a challenge here. It’s a preference.

---

### Dessert Adjacent Realist

You’re not eating dessert constantly.

You just keep ending up suspiciously close to it.

---

### Freshness With Boundaries

You like bright, clean flavors.

Just not so much that dinner starts feeling punitive.

## Description Structure

Each archetype description must be **3–4 sentences total**.

Sentence 1:
Describe the user's general food personality.

Sentence 2:
Add a dry or observational line about how they choose food.

Sentence 3 (optional but recommended):
Encourage the user to upload more dishes so the profile sharpens.

## Hard Writing Rules

The archetype description should **never list specific dishes**.

Avoid sentences like:
- "You keep coming back to ramen, sushi, and burgers."
- "Your favorites include tiramisu, cheesecake, and brownies."

Listing foods reads robotic and generated.

Instead describe **patterns and tendencies**:
- curiosity vs comfort
- repeat vs exploration
- bold vs balanced
- indulgent vs fresh
